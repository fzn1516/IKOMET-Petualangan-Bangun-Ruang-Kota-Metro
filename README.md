# IKOMET-Petualangan-Bangun-Ruang-Kota-Metro
IKOMET adalah game petualangan yang mengajak pemain mengikuti perjalanan karakter utama bernama Iko untuk menjelajahi lima ikon Kota Metro. Pada tiap ikon, pemain mengidentifikasi bangun ruang, menebak bentuk melalui interaksi klik, membaca materi, lalu menjawab kuis untuk memperoleh skor.
